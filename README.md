# WTI Prediction

This repository contains the smart contract source code for WTI Prediction. 
### Installation
```sh
# Clone the repo
    git clone https://github.com/danushkaadikari/WTIPrediction
# Install all dependencies
    npm install
```
### Compile contract
```sh
npx hardhat compile
```
### Test contract
```sh
npx hardhat test
```
### Deploy contract on testnet
```sh
npx hardhat run scripts/deploy-testnet.js --network bscTestnet
```
### Verify contract on testnet
```sh
npx hardhat run scripts/verify.js --network bscTestnet
```
